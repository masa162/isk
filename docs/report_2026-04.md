# ISK フロントエンド改修レポート 2026-04

## 作業日
2026-04-20

## 背景
記事アイキャッチ画像がなくChromeの新規タブ等のOGPピックアップに乗らない問題と、CSS・UX面での細かい未対応事項を一気に解消した。

---

## 実施した改修

### 1. CI/CD整備（重要）
**問題:** `public/styles.css` が `.gitignore` に入っていてgit管理外だったため、CSSの変更がデプロイに反映されなかった。

**対応:**
- `.github/workflows/deploy.yml` を新規作成
- GitHub Actions で `npm ci → npm run build:css → wrangler deploy` を自動実行
- `public/_headers` を追加して `styles.css` の Cloudflare CDN エッジキャッシュを無効化（`no-cache, no-store`）

**必要なGitHub Secrets:**
- `CLOUDFLARE_API_TOKEN`：Cloudflare Dashboard → My Profile → API Tokens → Edit Cloudflare Workers テンプレートで発行

---

### 2. アイキャッチ画像（OGP対応）
**対応:**
- D1 DB に `image_url TEXT` カラムを追加（`wrangler d1 execute --remote`）
- `Article` 型・`ArticleRepository`（create/update）・管理画面フォームに `image_url` フィールドを追加
- 記事カードにサムネイル表示（16:9、ホバーズーム）
- 記事詳細ページの OGP `og:image` / `twitter:image` を記事画像で上書き

**運用:** 管理画面編集フォームの「アイキャッチ画像URL」にCDN URL（`https://img.tokyo86.com/xxxxx/001.webp`）を入力。

---

### 3. テキストリンク色
`.prose a` のスタイルがなかった（グローバルリセット `* { margin:0; padding:0 }` で消えていた）。

**対応:** `input.css` に追加
```css
.prose a { color: #0066cc; text-decoration: underline; }
.prose a:visited { color: #6600cc; }
.prose a:hover { color: #0052a3; }
```

---

### 4. Markdownリスト表示
`.prose ul/ol` に `list-style-type` がなく、箇条書きが記号なしで表示されていた。

**対応:**
```css
.prose ul { list-style-type: disc; margin-left: 30px; }
.prose ol { list-style-type: decimal; margin-left: 30px; }
.prose li { margin-bottom: 6px; }
```

---

### 5. カテゴリクリックで一覧遷移
記事詳細の `<span class="category">` を `<a href="/articles?category=...">` に変更。

---

### 6. 下書きプレビュー
未公開記事（`published=0`）が以前は404だったところを、URL直打ちで閲覧可能に変更。公開一覧には出ない。記事上部に「⚠ これは下書き記事です」バナーを表示。

---

### 7. ページネーション
TOPページ・記事一覧ページに20件ページネーションを追加。`?page=N` パラメータで制御。

---

### 8. Mermaid記法対応
` ```mermaid ` ブロックを `<div class="mermaid">` に変換し、Layout.tsx で記事ページのみ Mermaid.js（ESM CDN）を読み込む。

**注意:** 2026-04-20 時点では表示確認ができていない（CSP or タイミング問題の可能性）。

---
